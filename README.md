# Interactive Literacy Hub

## Run locally
Open the folder in Visual Studio Code and use the Live Server extension on `index.html`.

## Upload to hosting
Upload all files and folders to `public_html`, `www`, or `htdocs`. Keep the structure unchanged.

## Included
11 responsive pages, original SVG images, quizzes, drag-and-drop vocabulary, tabs, local progress, printable resources, family activities, teacher modal, and APA-style references.

## Before final thesis submission
Review district privacy requirements, replace sample texts with final approved texts where needed, and ensure the reference list exactly matches the written thesis.
